## 開発

### ブランチ規則
- `feature/#<issueNo>-<description>`
- `fix/#<issueNo>-<description>`
- `refactor/#<issueNo>-<description>`
- `docs/#<issueNo>-<description>`
- `chore/#<issueNo>-<description>`

### コメントprefix

- `feat` - 機能追加
- `fix` - バグ修正
- `docs` - ドキュメント
- `style` - linter, formatterによる変更
- `refactor` - リファクタリング
- `perf` - パフォーマンス改善
- `test` - テストのみ変更
- `build` - ビルド・CIの変更
- `ci` - CI設定の変更
- `chore` - その他
